## 👬 기여하기

로드맵은 [발사믹](https://balsamiq.com/products/mockups/)을 사용하여 제작되었습니다.
프로젝트 파일은 `/project-files` 디렉토리에 있습니다. 로드맵을 수정하려면 발사믹을 열고 **Project > Import > Mockup JSON** 클릭하여 로드맵을 열고 수정한 후 이미지를 readme 에 업로드한 후 `Pull Request`를 작성합니다.

* 개선사항은 pull request
* 논의사항은 issues
