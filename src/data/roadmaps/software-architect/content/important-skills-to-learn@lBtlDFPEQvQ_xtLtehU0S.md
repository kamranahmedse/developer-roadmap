# Important Skills

To support the laid-out activities specific skills are required. From my experience, read books and discussions we can boil this down to these ten skills every software architect should have:

- Design
- Decide
- Simplify
- Code
- Document
- Communicate
- Estimate
- Balance
- Consult
- Market
