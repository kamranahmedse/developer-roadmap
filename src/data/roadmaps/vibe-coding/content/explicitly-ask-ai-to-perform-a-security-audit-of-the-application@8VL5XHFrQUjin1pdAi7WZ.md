# Explicitly ask AI to perform a security audit of the application

Before you launch anything to the public, ask the AI to go through your codebase and look for security vulnerabilities. Ask it specifically to check for common issues like exposed API keys, unprotected routes, missing input validation, insecure data storage, and anything else that could be exploited.
