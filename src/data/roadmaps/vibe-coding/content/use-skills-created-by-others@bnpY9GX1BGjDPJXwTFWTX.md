# Use skills created by others

There is a growing open ecosystem of skills built by developers, and many AI tools are starting to support a shared standard that makes these skills plug-and-play across different platforms. Instead of spending time figuring out how to get your AI to do something from scratch, chances are someone has already built a skill for it.
