# Bun

Bun is an all-in-one toolkit for developing modern JavaScript/TypeScript applications. It is made of a runtime designed as a replacement of Node.js, a package manager, a test runner and a bundler.
  
Visit the following resources to learn more:

- [@official@Bun Website](https://bun.sh/)
- [@official@Build your first app with Bun](https://bun.com/docs/quickstart)
