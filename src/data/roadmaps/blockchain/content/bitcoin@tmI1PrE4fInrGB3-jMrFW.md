# Bitcoin

Bitcoin is a decentralized digital currency, meaning it operates without a central bank or single administrator. Transactions are verified by network nodes through cryptography and recorded in a public, distributed ledger called a blockchain. Bitcoin uses a UTXO (Unspent Transaction Output) model, where each transaction creates new UTXOs that can be used as inputs in future transactions, similar to how physical cash works.
