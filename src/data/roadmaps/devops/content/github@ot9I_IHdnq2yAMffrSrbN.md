# GitHub

GitHub is Microsoft's web-based platform for hosting Git repositories, offering collaborative features like pull requests, issues, GitHub Actions CI/CD, and project boards. For DevOps engineers it is the central hub for source control, code review, and triggering automated deployment pipelines.
