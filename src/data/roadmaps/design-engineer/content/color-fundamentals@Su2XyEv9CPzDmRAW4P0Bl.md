# Color Fundamentals

Color communicates meaning, creates mood, and guides attention. Design Engineers must understand color theory and implement accessible, systematic color palettes.

**Color Relationships**: Complementary (opposite on wheel), analogous (adjacent), triadic (triangle). Understanding these creates harmonious palettes.

**Building Palettes**: Start with 1-2 primary colors, add neutrals (grays), then semantic colors (success green, error red, warning yellow). The 60-30-10 rule distributes dominant, secondary, and accent colors.

**Accessibility**: WCAG requires 4.5:1 contrast ratio for normal text, 3:1 for large text. Never rely on color alone to convey information. Test with contrast checkers.

**Implementation**: Use CSS custom properties for color tokens. Define primitives (blue-500) and semantic tokens (color-primary) separately. Plan for dark mode from the start.

**Modern Color Spaces**: oklch and oklab provide perceptually uniform color manipulation. Better for creating consistent color scales.
