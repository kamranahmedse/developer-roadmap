# Technical Interview Preparation

Design Engineer interviews often combine frontend technical skills with design sensibility assessment. Preparation should cover both coding ability and design thinking.

**Technical Screen Topics**:
- Vanilla JavaScript (DOM manipulation, events, array methods)
- React fundamentals (hooks, state, props, lifecycle)
- CSS layout (Flexbox, Grid, responsive design)
- TypeScript basics
- Build a component from scratch

**Common Coding Challenges**:
- Implement a UI component (tabs, accordion, modal)
- CSS layout challenges
- State management scenarios
- Accessibility improvements

**Design/Product Questions**:
- Critique a UI design
- Propose improvements to an interface
- Discuss trade-offs in implementation approaches
- Explain your design decisions from portfolio projects
