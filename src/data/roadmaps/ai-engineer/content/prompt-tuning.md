# Prompt Tuning

Prompt tuning is a method for adapting large AI models to new tasks without changing the model itself. Instead of retraining billions of parameters, it learns a small set of special vectors called “soft prompts” that are added to the input. These act like hidden instructions that guide the model toward the desired behavior while keeping the original model unchanged. This approach is much more efficient in terms of computation and storage, making it a practical choice for customizing models across many different tasks.
