# Reverse Linked List

Given the head of a linked list, reverse it in place and return the new head. You iterate through the list keeping track of the previous node, current node, and next node, rewiring each pointer as you go. This is the first linked list problem most people learn and it teaches you the three-pointer technique that underlies almost every in-place list manipulation. Visit the question on the LeetCode [website](https://leetcode.com/problems/reverse-linked-list/).
