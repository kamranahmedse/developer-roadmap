# Valid Palindrome

Given a string, determine if it reads the same forwards and backwards after removing non-alphanumeric characters and ignoring case. Two pointers start at each end and move inward, comparing characters as they go. This problem teaches you to use two pointers on a string and is a clean entry point for understanding how pointers can replace nested loops. Visit the question on the LeetCode [website](https://leetcode.com/problems/valid-palindrome/).
