# Swim in Rising Water

Given a grid where each cell has a height, find the earliest time t such that you can travel from top-left to bottom-right, moving only through cells with height at most t. You binary search on t or use Dijkstra treating each cell's height as the cost. This problem teaches you to reframe a graph problem as a min-max path problem, where you minimize the maximum cost along any path. Visit the question on the LeetCode [website](https://leetcode.com/problems/swim-in-rising-water/).
